### 아나콘다 다운로드 
https://www.anaconda.com/download/#macos

### 콘다 가상환경 리스트
```shell
conda info --evn
```

### 콘다 가상환경 생성
`conda create --name exam2 python=3.6 anaconda`

### 가상환경 스위치
`source activate exam2`

### conda tensorflow install
`conda install tensorflow`
